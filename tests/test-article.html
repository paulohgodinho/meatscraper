Written by John Doe on February 5, 2024

This is a test article to demonstrate the capabilities of the meatscraper extraction tool. The extraction process will identify and extract the main content from this page.

Section 1: Key Features

meatscraper provides several important features:

  • Metadata extraction from HTML headers
  • Main content identification using readability algorithms
  • HTML sanitization for safe content
  • Plain text conversion with formatting preserved
  • Primary image selection from metadata

Section 2: How It Works

The extraction process happens in 5 steps:

  1. Extract metadata using metascraper
  2. Identify readable content with Mozilla Readability
  3. Sanitize HTML using DOMPurify
  4. Convert to plain text
  5. Select primary image

This multi-step pipeline ensures high-quality content extraction from complex web pages.
